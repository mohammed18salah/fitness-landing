# ⚡ PULSE — Premium Fitness & Workout

PULSE is a modern, high-performance premium fitness landing page and interactive workout generator. Built with **React 19**, **Vite**, and **Tailwind CSS**, it features a fully responsive, gorgeous glassmorphic interface and a smart workout split builder.

---

## ✨ Features

- **⚡ Smart Workout Generator**: Instantly generate personalized workout splits based on your goals (Muscle Building, Fat Loss, Strength, General Health), fitness level, and availability.
- **📚 1,300+ Exercise Library**: Uses a rich dataset of real exercises with categorization (chest, back, cardio, etc.) and equipment specifications.
- **💎 Premium Glassmorphism UI**: Beautifully designed dark-mode interface with smooth gradients, custom neon lighting effects, and animations.
- **📱 Fully Responsive**: Tailored to look stunning and work seamlessly on mobile, tablet, and desktop screens.
- **🚀 Ultra-fast Build**: Uses Vite 8 for blazing fast hot module replacement (HMR) and optimized Rolldown production builds.

---

## 🛠️ Technology Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 8](https://vite.dev/)
- **Styling**: [Tailwind CSS v3](https://tailwindcss.com/)
- **Linter**: [Oxlint](https://oxc.rs/) (Extremely fast JavaScript/TypeScript linter)

## 🚀 Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed (version 18 or higher is recommended).

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mohammed18salah/fitness-landing.git
   cd fitness-landing
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

### Development Server

Start the local development server:
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:5173`.

### Production Build

To build the project for production:
```bash
npm run build
```
The output files will be generated in the `dist/` directory, optimized and ready for hosting.
